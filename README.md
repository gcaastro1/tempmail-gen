## 📖 About

**DropMailer** is a web application designed to generate temporary, disposable email addresses. It allows users to protect their personal information, avoid spam, and receive incoming emails directly within the application in real-time. 

Built with a focus on performance and a clean user experience, the application interfaces with the robust Dropmail GraphQL API.

## ✨ Features

- **Instant Email Generation:** Get a ready-to-use temporary email address instantly.
- **Real-Time Inbox:** Automatically receive incoming emails without needing to refresh the page.
- **Privacy First:** Keep your primary inbox clean and safe from spam, newsletters, and unwanted subscriptions.
- **Responsive Design:** Beautiful and intuitive user interface tailored for both desktop and mobile devices.

## 🛠️ Technologies Used

- **Framework:** [Angular](https://angular.io/) (v16)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [TailwindCSS](https://tailwindcss.com/)
- **Data Fetching:** [Apollo GraphQL](https://www.npmjs.com/package/apollo-angular)
- **Icons:** [FontAwesome](https://fontawesome.com/)
- **API Provider:** [Dropmail.me](https://dropmail.me/api/#)

## 🚀 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

Ensure you have Node.js and npm installed.
- [Node.js](https://nodejs.org/en/download/) (v16+ recommended)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/tempmail-gen.git
   cd tempmail-gen
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm start
   ```

4. **Access the application:**
   Open your browser and navigate to [http://localhost:4200/](http://localhost:4200/). The app will automatically reload if you change any of the source files.
